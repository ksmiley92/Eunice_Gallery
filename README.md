# Eunice's Gallery

Photography gallery for Eunice Guevara. Displays printable fine art photos
available for purchase via Stripe.

## Stack
- HTML5 / CSS
- Cormorant Garamond + Source Sans 3 (Google Fonts)
- Stripe Buy Buttons

## Pages
- `index.html` — gallery
- `contact.html` — contact

## Versions
- **v0.1** — Original layout
- **v0.2** — Design refresh, PDF links, Stripe integration
- **v0.2.1–v0.2.8** — Layout changes and small bug fixes
